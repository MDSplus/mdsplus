Classes
Compound
Action
An Action is used for describing an operation to be performed by an MDSplus action server.
Call
A Call is used to call routines in shared libraries.
Conglom
A Conglom is used at the head of an MDSplus conglomerate.
Dependency
A Dependency object is used to describe action dependencies.
Dimension
A dimension object is used to describe a signal dimension, typically a time axis.
Dispatch
A Dispatch object is used to describe when an where an action should be dispatched to an MDSplus action server.
Function
A Function object is used to reference builtin MDSplus functions.
Method
A Method object is used to describe an operation to be performed on an MDSplus conglomerate/device
Procedure

Program

Range

Routine

Signal
A Signal is used to describe a measurement, usually time dependent, and associated the data with its independent axis (Dimensions).
Window
A Window object can be used to construct a Dimension object.
Opaque
